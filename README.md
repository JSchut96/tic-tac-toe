# Tic-Tac-Toe

A simple Tic-Tac-Toe project with multiple multiplayer options and two different game modes.

## Features

### Multiplayer

* Local play (same device)
* Local network play (LAN)
* Internet play

### Game Modes

**Classic**

* Standard Tic-Tac-Toe rules.
* First player to get three in a row wins.

**Decay**

* Each player can only have **3 active moves** on the board.
* When placing a 4th move, the oldest move is removed.
* This keeps the board changing throughout the game and adds an extra layer of strategy.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/JSchut96/tic-tac-toe.git
cd tic-tac-toe
python3 main.py
```
